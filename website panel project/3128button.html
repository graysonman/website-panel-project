Power Series Pro HS3128

LCD Keypad (Button)

Service

Arm or Disarm
  • To arm, press [*][0]. Or press the home icon for 2 seconds plus user access code.
  • To disarm type in user code.
Bypass Sensor
  • On the keypad, press [*] [1].
  • To bypass a zone, choose one of the following options:
  • Enter a three digit zone number
  • Use the [<] [>] keys to scroll to the zone, and press [*].
  • To exit Bypassing mode, press [*].
Silence troubles
  • Press the [*] [2] key to silence.
  • Note: If troubles come and go the beeping will resume
Reset Sensors
  • Press [*] then [7] then [2]
  • Note: There are 2 different reset buttons that will work as well. Hold for 3 seconds to do the reset. There is a labeled reset button and a button that looks like

Install

Enable DLS Communication
  • Press * [8]
  • Enter Installer code.
  • Option [401] then option [7]
Access Code
  • Press * [8]
  • Enter Installer code.
  • Option [403] then enter panel model followed by 00
Alarm.com Enable
  • Press * [8]
  • Enter Installer code.
  • Option [382] then option [5]
Communication Test
  • Press * [8]
  • Enter Installer code.
  • Option [852] then scroll to Communication Test or Cellular Test and hit *
Enroll Modules
  • Press * [8]
  • Enter Installer code.
  • Option [902] then choose auto enroll by hitting [*]
Differences in keypad model: ENG has english description buttons.
RFPRO9 and WFPRO9 have PowerG transceivers built in.
