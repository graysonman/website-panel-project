Power Series Pro HS3128

Touchscreen Keypad

Service

Arm or Disarm
  • Touch the top bar in the middle of the screen.
  • It will start the arming process if system is not armed.
  • If the system is armed enter the user code to disarm.
  • If there are alarm memory conditions acknowledge them by arming and disarming.
Bypass Sensor
  • Click on options then scroll to bypass sensors.
  • Click on zones to bypass.
Silence troubles
  • Click the troubles button.
  • Note: If troubles come and go the beeping will resume
Reset Sensors
  • Go to options then keypad mode.
  • Press [*] then [7] then [2]

Install

Enable DLS Communication
  • Go to Settings then go to Keypad Mode.
  • Press * [8]
  • Enter Installer code.
  • Option [401] then option [7]
Access Code
  • Go to Settings then go to Keypad Mode.
  • Press * [8]
  • Enter Installer code.
  • Option [403] then enter panel model followed by 00
Alarm.com Enable
  • Go to Settings then go to Keypad Mode.
  • Press * [8]
  • Enter Installer code.
  • Option [382] then option [5]
Communication Test
  • Go to Settings then go to Keypad Mode.
  • Press * [8]
  • Enter Installer code.
  • Option [852] then scroll to Communication Test or Cellular Test and hit *
Enroll Modules
  • Go to Settings then go to Keypad Mode.
  • Press * [8]
  • Enter Installer code.
  • Option [902] then choose auto enroll by hitting [*]
